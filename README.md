# Vision Board Creator

A beautiful, interactive web application to create personalized vision boards with images that inspire you.

## Features

- **Image URL Input with Live Preview**: Paste image URLs and see a preview instantly before adding to your board
- **Smart Canvas Layout**: Images automatically fit within the canvas - add 1 to 12+ images and they'll be arranged in an optimal grid
- **Drag and Resize**: Move images around by dragging, and resize them using the corner handle
- **Export Options**: Download your vision board as:
  - PNG (high quality)
  - JPG (compressed)
  - PDF (document format)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Usage

1. Paste image URLs in the input fields
2. See the preview of each image immediately
3. Add more images using the "+ Add Another Image" button
4. Click "Create Visionboard" to generate your board
5. Drag images to reposition them on the canvas
6. Use the corner handle to resize images
7. Click "Export" to download in your preferred format

## Tech Stack

- React 19
- Vite
- html2canvas (for canvas capture)
- jsPDF (for PDF export)
